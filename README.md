# Number Guessing Game 🎮

A beginner Python project where the computer picks a random
number and the user guesses it. Built to practice loops,
conditionals, and user input in Python.

## How to run

```bash
python game.py
```

## Features
- Generates a random number between 1 and 100
- Gives "Too high" or "Too low" hints after each guess
- Counts and displays total number of attempts
- Simple and clean terminal interface

## What I learned
- How to use Python's `random` module
- Writing while loops and break conditions
- Taking and validating user input
- Basic Git and GitHub workflow (my first repo!)

## Built with
- Python 3.x
- Random module (built-in)
